## 🏃 Три команды для старта

### Вариант 1: CodeSandbox
```
1. Загрузите проект на GitHub
2. CodeSandbox.io → Import from GitHub
3. Готово!
```

### Вариант 2: Локально
```bash
cd kadastr-game
npm install
npm run dev
```

### Вариант 3: Production
```bash
npm run build
# → папка dist/
```
